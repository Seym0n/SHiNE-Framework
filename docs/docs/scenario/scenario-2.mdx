# Scenario 2

In the Scenario 2 which you typically see when you spin up the base Docker (Compose) image, we have three rooms:
- a living room
- Bob's Room
- Your Room

Among these rooms, the user need to complete these three tasks

| # | Task | Feasible |
| 1 | Turn on the Deep Fryer | Yes |
| 2 | Turn on the Mixer | No |
| 3 | Open Book in your own room | Yes |

Each task is accompanied by a smart home rule which is designated as follow 

| # | Rule | Associated Task |
| 1 | Deep Fryer Only On When Cooker Hood Is On | Task 1 |
| 2 | Turn Off Mixer After 22:00 | Task 2 |
| 3 | Turn On Lamp When Book Is Open After 3 Seconds | Task 3 |
